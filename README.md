# NLP Chatbot using Python

## Overview
This project is a simple Natural Language Processing chatbot built using Python and the NLTK library.  
The chatbot can respond to user inputs using predefined conversational patterns and basic NLP techniques.

## Features
- Interactive chatbot conversation
- Pattern matching using NLTK
- Natural language preprocessing
- Simple conversational AI responses

## Technologies Used
- Python
- NLTK (Natural Language Toolkit)
- Jupyter Notebook

## How to Run the Project

1. Clone this repository

```
git clone https://github.com/YOUR_USERNAME/nlp-chatbot.git
```

2. Install required libraries

```
pip install nltk
```

3. Run the notebook

Open **NLP_Chatbot_Project.ipynb** in Jupyter Notebook and run all cells.
